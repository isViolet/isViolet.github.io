# 控制图片的移动切换

没有对 Player 进行图片的设置当然图片也就不会改变了，你切好后的多个图片，需要通过自己的小手在 player 脚本里写好所需要切换图片的代码才行。

下面简单说明下如何进行编写☟

  • 通过改变该图片的转向
  • 改变不同转向图片达到转向的效果

# 改变图片的转向

思路：坦克大战本身是 2D 游戏，但是他依然是有 z 轴的，所以可以改变 z 轴的旋转角度改变图片的显示。

①渲染器引用

//设置精灵渲染器 
private SpriteRenderer sr; 

//在awake方法中添加语句
private void awake(){//☆脚本下没有awake方法手动添加
    //得到使用该脚本的预制体的精灵渲染器
    sr = GetComponent();
}

②根据 h 和 v 的值，确定图片的转向

//用于存储上下左右四张图片
private Sprite[] tankSprite; 

//h是横轴方向，h=1代表向右,h=-1代表向左
float h = Input.GetAxisRaw("Horizontal");
transform.Translate(Vector3.right*h*moveSpeed*Time.deltaTime,Space.World);
if(h<0){
    sr.sprite = tankSprite[3];//这里的tankSprite[3]是方向向右的图片
}else if(){
    sr.sprite = tankSprite[4];//同理
}

//v代表垂直方向，与h是相同的操作。
