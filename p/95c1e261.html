# 顺序表的插入操作

主要操作步骤及其实现方法：

  • 检测（判断参数 i 是否合理及空间是否满）
    1. 若 i<1||i>L.length+1，插入位置不合理，算法结束；
    2. 若 L.length>=L.listsize，则存储空间已满，需增加分配空间
  • 后移（插入位置及之后的所有元素后移一位）
q = &(L.elem[i-1]); //q指示插入位置
for(p = &(L.elem[L.length-1]);p>=q;--p)
\*(p+1) = *p; //后移一位
  • 插入（将新元素插入到指定位置上）
*q = e; //插入e
  • 修正表长（表长加 1）
++L.length; //表长增1

# 顺序表的删除操作

主要操作步骤及其实现：

  • 检测（判断参数 i 是否合理）若 i<1||i>L.length，删除位置不合理，算法结束
  • 前移（删除元素之后的所有元素前移一位）
p = &(L.elem[i-1]); //p指示删除位置
e = *p; //用e返回被删除元素的值
q = &(L.elem[L.length-1]); //p指示表尾位置
for(p++; p>=q; --p)
    *(p+1) = *p; //后移一位
  • 修正表长（表长减 1）
--L.length; //表长减1
