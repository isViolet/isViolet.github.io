# cd 命令

cd：切换目录
pwd：当前目录
cd -：回到前一次目录
cd ..：回到上一级目录
cd ../..：回到上一级目录的上一级目录
除非长期工作在该目录，否则尽量少用 cd

# ls 及常用参数

# 文件权限和所有者

drw-r--r--
d：表示文件夹
前三位：所有者的权限
中间三位：所有者所在组的其它用户的权限
后三位：其他组的权限
r：可读 4
**w：可写 2 **
x：可执行 1

# cp、rm、mv、多敲回车

cp：拷贝文件
cp -r：拷贝文件及目录
rm：删除文件
rm -r：删除目录及文件
rm -rf：强制删除

移动 123 文件到当前目录并取名 333
mv /182/123 ./333

# 根目录及其成员粗略介绍

# mkdir 和 touch

mkdir：创建空目录
touch：创建空文件
cat，more：查看文件内容

# vi、vim 作用及常见按键


vim 非编辑下
大写的 I：光标移到改行的最左侧
o：光标向下插入一行
大写的 O：向上插入一行
gg：光标跳到文件最顶部
G：光标跳到文件最底部
yy：复制该行
p：粘贴
dd：删除该行
u：撤销
x：删除某个字符
s：删除某个字符并且进入编辑模式
